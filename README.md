# Volunteer Hub — корпоративное волонтёрство Chevron в Казахстане

Хакатон «Startup Platform для партнёра», июль 2026.

## Партнёр

**Employee Volunteer Program (NVS & Zhurekten Zhurekke)** — программа корпоративного волонтёрства компании Chevron в Казахстане. Включает субботники, посадку деревьев, мероприятия поддержки детей с инвалидностью и из малообеспеченных семей. Города: Астана, Алматы, Атырау.

## Проблема

Сейчас записи на волонтёрские акции ведутся в рабочих рассылках и Excel-таблицах. Координатор не видит в реальном времени, кто и сколько человек записалось, сотрудники не знают, какие события доступны.

## Решение

Volunteer Hub — веб-приложение, где координатор публикует волонтёрские события, а сотрудники записываются на них в 1 клик.

### Возможности

1. **Список событий** с категориями, городами, датами и количеством участников
2. **Запись в 1 клик** — имя + контакт, счётчик обновляется мгновенно
3. **Остаток мест** — видно, сколько свободно из общего числа; если мест нет — кнопка блокируется
4. **Фильтр по городам** — Астана, Алматы, Атырау
5. **Панель координатора** — просмотр списков участников по каждому событию, копирование в буфер
6. **Кнопка «Поделиться»** — ссылка на событие с якорем
7. **Форма отзыва о платформе** — сбор фидбэка для хакатона
8. **RU/KZ** — переключатель русского и казахского языков

### Продакшн

**[volunteer-hub-kappa.vercel.app](https://volunteer-hub-kappa.vercel.app)**

## Стек

- Статический сайт: HTML + vanilla JS
- База данных: Supabase (PostgreSQL)
- Supabase JS-клиент через CDN
- Деплой: Vercel

## Запуск

**Продакшн (рекомендуется):** открыть ссылку Vercel (см. URL деплоя).

**Локально:**
```
python3 -m http.server 8080
```
Открыть http://localhost:8080 в браузере.
